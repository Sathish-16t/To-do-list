#to do list web aplication

a simple to do list built with HTML,CSS and Javascript

#features

-create tasks
-delete task
-mark completed task
-search task
-theme(dark & light)
-saves task using localstorage

#how to use
1.type a task in the input box
2.hit the create button
3.click the checkbox if task is comleted
4.use delete button to delete the task
5.you can also search task by its name
